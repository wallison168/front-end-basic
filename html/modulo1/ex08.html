Utilizando links externos

voce pode acessar o eu repositorio : github
Obs: ao utilizar links externo e ideal definir 2 parametros para melhor compreenção do codigo
1º target : utilizando o parametro "target" com o valor "_blank" ao invez do link sobrescrever a pagina atual ou seja sair da pagina atual para ir ao destino do link ele ira abrir o endereço do link em outra pagina
2º rel : nesse parametro voce ira definir o tipo do link se ele e um link externo ou interno nesse caso por se tratar de link externo ele recebe o valor "external"

Ex:
links externos sem parametros "target, rel" : curso em video
links externos com o uso dos parametros "target, rel" : cfb cursos

Utilizando liks internos

acessando link interno para produzir : imagens
nesse caso definimos o valor do parametro "rel" como "next" ou seja nesse caso o navegador ira identificar que ao clicar nesse link ele ira direcionar o usuario para a proxima pagina

Utilizando links para download

basicamente o link para download e similar aos demais links, com isso basta fazer a chamado do arquivo para download
Ex : baixe aqui : pdf css3
pode ocorrer casos do navegador não começar o download mas sim abrir o arquivo diretamente no navegador, caso aconteça isso e indicado mandar o aquivo de download zipado
Ex2: baixe aqui : pdf css3
para que haja corretamente o download e nescessario definir o tipo do arquivo nesse caso colocamos o valor dentro do "type" "application/zip" e colocar o parametro "download" com o nome do arquivo assim como no exemplo acima
