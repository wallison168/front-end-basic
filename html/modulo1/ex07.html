Trabalhando com Listas

Lista Ordenada

Basicamente uma lista ordenada se trata de tarefas a serem executadas que possue uma ordem certa de execução nesse caso utilizamos essa modelo de lista caso ten-se a nescessidade de acresentar mais alguma tarefa sem quebrar a orem da lista, para definirmos a lista ordenada e nescessario utlizarmos a tag "ol" basicamente essa tag ira definir a ordem da lista. Os itens da lista devem ser armasenados dentro da tag "li"
Ex:

    Carros

  1. Gol
  2. Celta
  3. Palio
  4. Uno
  5. Corola
  6. Cruze

Observa-se que utilizamos a tag "ol" para ordenar os itens da lista e utilizamos a tag "li" para definir os itens da lista
Obs: pode se mudar o metodo de ordenagem da lista alterando o parametro "type" dentro da tag 'ol' ou seja com isso ao invez da tag "ol" gerar uma ordenação numerica podemos definir outros padrões
Ex:

    Motos

  1. Titã
  2. XJ6
  3. XRE
  4. Broz

Observa-se que o parametro "type" da tag "ol" foi alterado, com isso acabou gerando uma alteração na forma de ordenação da lista

Lista não ordenada

Basicamente a lista não ordenada e similar a lista ordenada, o seu diferencial e que com ela definimos tarefas a serem executadas porem não existe uma ordem certa de execução, para definirmos essa lista ao invez de se utilizar a tag "ol" utilizamos a tag "ul"
Ex:

    Alimentos

  • Pão
  • Carne
  • Queijo
  • manteiga

Obs: Assim como na tag "ol" podemos modificar o "type" da tag "ul" tambem

Lista Aninhadas

Basicamente a lista aninhada se trada de uma lista comum (ordenada ou não ordenada) que possue outra lista dentro dela mesma
Ex:

    Linguagens de programação

  1. Antigas
    • C#
    • Python
    • C++
  2. Novas
    • JavaScript
    • Node.js
    • Vue.js

Observa-se que colocamos uma lista dentro da outra ou seja nesse caso criamos uma lista aninhada

Lista de Definição

Basicamente uma lista de definição e como se fosse um dicionario, colocamos um termo dentro dessa lista e logo em seguida colocamos a definição desse termo, para definirmos a lista de definição e nescessario colocar a tag "dl" essa tag que ira definir a lista, para inserir um termo dentro dessa lista deve-se colocar o termo dentro da tag "dt", e para definirmos a definição
Ex:

Html
Linguagem de marcação para criação de conteudo de um site
Css
Liguagem de formatação para conteudos de um site

Observa-se que definimos um termo dentro da tag "dt" e logo em seguida colocamos a definição desse termo dentro da tag "dd"
