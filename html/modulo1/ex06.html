Codigo fonte

Basicamente ao utilizar essas tags Html isso permite que o navegador possa renderizar codigos de outras linguagens ou seja nesse caso ele não irar gerar a quela funcionalidade definida porem a visibilidade do codigo sera de mehlor compreensão

como no exemplo a baixo utilizaremos a tag "code" com essa tag teremos uma visibilidade melhor dos elementos inserido pois ela gera uma formatação que permite que todos os caracteres tenham o mesmo espaço definido
Ex :
1º sem o uso da tag "code" : document.getElementeById('teste')
2º com o uso da tag "code" : document.getElemeteById('teste2')

Obs: a tag "code" formata a font da texto definido para permitir que os espaçamentos sejam considerado e nescessario utilizar a tag "pre" para validar essa funcionalidade.
