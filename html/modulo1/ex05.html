Principais formatações

Utilizando o negrito

Podemos utilizar o negrito de três formas diferentes, duas dessas formas são definidas atravez do proprio Html ja a outra forma e definida atravez do css
1º utilizando o negrito atravez da tag "b" Negrito
2º utilizando o negrito atravez da tag "strong" Negrito

Obs: Observa-se que ambas as formas traz o mesmo resultado porem oque muda e a semantica pois a tag "b" reliza somente o efeito de negrito ja a tag "strong" realiza o efeito de destaque ou seja um elemento forte, importante

Utilizando o italico

Podemos utilizar o italico de tres formas, duas dessas formas e atravez do Html ja a outra e atravez do css
1º utilizando o italico atravez da tag "i" Conteudo em Italico
2º utilizando o italico atravez da tag "em" Conteudo em Italico

Obs: ja no caso do italico ele possue duas formas de ser definido atravez do html, similar ao negrito ou seja utilizar o italico atravez da tag "i" realiza somente o efeito italico ou seja somente a formatação ja utilizando a tag "em" da enfaze pro conteudo ou seja tem uma semnatica mais forte

Utilizando o marcador de texto

Basicamente podemos marcar um texto utilizando a tag "mark" nesse caso essa tag ira marcar o texto como se estivesse passado um marcado de texto
1º Utilizando a tag "mark" testo marcado

Obs: para mudar a cor da marcação provocada pela tag "mark" e nescessario realizar a formatação atravez do css

Utilizando o redutor de texto

Basicamente podemos definir partes do texto para ficarem pequenas utilizando a tag "small" utilizamos esse mecanismo quando se tem a nescessidade de se colocar partes importantes como contratos em letras menores o famoso "pequenas letras"
1º Utilizando a tag "small" Texto pequeno

Excluindo e Inserindo textos

Podemos marca um texto como excluido para indicar que ele deve ser lido porem não considerado
1º Utilizando a tag "del" Excluindo texto

Podemos marcar um texto como inserido para dar enfase e indicar que esse texto foi inserido depois
1º Utilizando a tag "ins" Inserindo texto
