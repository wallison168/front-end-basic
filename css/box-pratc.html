Divisão de um site

Basicamente um site e dividido por 3 tags principais na qual cada uma define a sua funcionalidade.
Atigamente definiamos as parates do site utilizando a tag "div" e passando um "id" como identificador para a funcionalidade que a tag iria aplicar, hojem em dia existem tags semanticas que realizam essa funcionalidade ou seja não ha mais a nescessidade de se encher o site de tags do tipo "div"

Tags que dividem o site

Basicamente existem 3 tags semanticamente que definem as 3 partes de um site

1º header : Basicamente essa tag define o cabeçalho do site ou seja essa tag define o menu do site.

2º main : Basicamente essa tag defini o corpo do site ou seja essa tag define o conteudo que sera inserido dentro site sendo ele o conteudo principal.

3º footer : Basicamente essa tag define o rodape do site ou seja os link ou conteudo sobre o desenvolvedor no final do site

Definições de um Cabeçalho

Basicamente a tag "header" define o cabeçalho da aplicação ou seja todo conteudo que envolve cabeçalho como ("menu", "links", "pesquisa", "logo" etc) e definido dentro dessa tag.

Dentro da tag "header" definimos a tag "nav" atraves dele definimos link de navegação ou seja links e menus de um cabeçalho e definido dentro dessa tag pois ela da um sentido semantico melhor dentro da pagina por se tratar de uma tag de navegação.

Definição do Corpo da Aplicação

Basicamente tag "main" define o corpo da aplicação ou seja todo o conteudo a ser mostrado pro usuario sera definido detro dessa tag, essa tag traz uma semantica pro navegador ja que todo conteuido inserido dentro dela tera um sentido de "conteudo principal".

Podemos separar o conteudo principal em seções ou seja separando o conteudo em topicos diferentes para realizar isso, temos que definir dentro da tag "main" a tag "section" atravez dela definimos cada seção que vai ser inserida dentro do corpo da aplicação.

Dentro da tag "section" podemos utilizar a tag "article", essa tag da um sentido pro conteudo inserido como se fosse um "artigo".

Dentro da tag "article" podemos utilizar a tag "aside" as informações inseridas dentro dessa tag faz referencia ao conteudo inserido dentro da tag "article" ou seja a todo conteudo inserido dentro da tag "aside" faz referencia ao conteudo inserido dentro da tag "article" ou seja a tag "aside" tem o sentido semantico de conteudo periferico ao conteudo da tag "article", em geral o conteudo inserido dentro da tag "aside" são conteudos referente ao artigo como ("nome do altor", "resulmo" etc) basicamente a tag "aside" faz refencias a outros conteudos.

Definição do Rodapé

A tag footer e similar a tag "header" o seu diferencial e que ela serve para definir o rodapé da aplicação ou seja dentro dele colocaremos os conteudos complementares do site como link e conteudo referente ao equipe de desenvolvedores e outros conteudos relacionados ao site.
