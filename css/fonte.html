Definindo o tamanaho da fonte

basicamente par definirmos o tamanho da fonte dentro do css basta chamar o parametro "font-size" atravez desse parametro podemos definr o tamanaho da fonte de diferenetes formas ou seja esse parametro pode receber varios valores que são capaz de definir um temanho ideal para a fonte
Obs: basicamente existe dois metodos diferentes de definir um tamanho de uma fonte ("medida-absoluta" "medida-realtiva")

  1. Medidas-Ordenadas
    • cm : centimetro
    • mm : milimetro
    • in : polegada
    • px : pixel
    • pt : ponto
    • pc : paica
  2. Medidas-Relativas
    • em : medida relativa do tamanho atual da fonte
    • ex : medida relativa a autora "x" de uma fonte
    • rem : medida relativa de acordo com o tamanho da fonte que foi definida dentro do "body"
    • vw : medida relativa referente a largura da view-pot ou seja relativaa largura do seu monitor
    • vh : e similar ao "vw" o seu diferencial e que ao invez de definir a medida relativa de acordo com a largura ele defini de acordo com a altura

Obs: para definir tamanho das fontes no css e recomendado utilizar dois padrões ("px" e "em").
O tamanho da fonte pre definido pelo proprio navegar e de "16px" sendo assim o valor "1em corresponde a == 16px"

Definindo o peso da fonte

Basicamente o peso da fonte se da pelo quantidade do contraste da fonte ou seja se a fonte vai ser mais leve ou se ele vai ser mais forçada
resumidamente o peso da fonte define se o texto sera mais fraco ou seja mais claro ou se ele sera mais forte ou seja com um auto teor de negrito
Obs: Existem 4 maneiras de definir o peso da fonte, para inserir o peso da fonte dentro do css e nescessario chamar o parametro "font-weight" basicamente atravez desse atributo podemos definir o peso do texto inseridom, esse parametro recebe 4 valores pra definir o peso da fonte cada valor define um peso diferente
1º lighter: basicamente esse valor definin uma fonte mais leve ou seja mais fraca
2º normal : basicamente esse e o peso padrão que as fontes vem no geral
3º bol : basicamente esse valor aplica um negrito na fonte ou seja deixa ele com um aspecto mais forte
4º bolder : basicamente esse valor e similar ao valor "bold" o seu diferencial e que ele aplica um negrito mais forte no texto
Obs: pode se definir o peso da fonte utilizando valores numericos de (100 - 900)

Impotante

Podemos definir toda formatação feita nos exemplos acima em uma unica linha de codigo
ao invez de utilizar varias linha um definindo uma formatação diferente da outra podemos apenas utilizar um linha de codigo para definir irsso
Ex : Formatação css com multiplas linhas de codigo 

    
        span{
            font-family:'Times New Roman', Times, serif;
            font-weight: bold;
            font-size: 2em;
            font-style: italic;
        }

Ex : Formatação css com uma linha de codigo
Obs = Para definir varias formatção em uma unica linha de codigo e nescessario seguir uma ordem de chama para cada parametro
Ordem de chama para fontes ("font-style" -> "font-weight" -> "font-size" -> "font-family")
Basicamente deve-se seguir a ordem dos parametros definida logo acima 
    
        font: italic bold 2em 'Times New Roman', Times, serif ;

Obs: observa-se que ambos os 2 exemplos realizam a mesma formatação porem o segundo exemplo possue a menor quantidade de linha de codigo
