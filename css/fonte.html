basicamente as fontes possuem 5 categorias mais usadas na web
1º serifadas
2º sans-serif
3º monoespaçadas
4º handwriting
5º display

Definindo um tipo de fonte

Obs: para definir um tipo de fonte dentro do css deve se chamar o parametro "font-family" atarvez desse parametros definis a tipagem da fonte a ser inserida dentro da pagina web

Definindo o tamanaho da fonte

basicamente par definirmos o tamanho da fonte dentro do css basta chamar o parametro "font-size" atravez desse parametro podemos definr o tamanaho da fonte de diferenetes formas ou seja esse parametro pode receber varios valores que são capaz de definir um temanho ideal para a fonte
Obs: basicamente existe dois metodos diferentes de definir um tamanho de uma fonte ("medida-absoluta" "medida-realtiva")

  1. Medidas-Ordenadas
    • cm : centimetro
    • mm : milimetro
    • in : polegada
    • px : pixel
    • pt : ponto
    • pc : paica
  2. Medidas-Relativas
    • em : medida relativa do tamanho atual da fonte
    • ex : medida relativa a autora "x" de uma fonte
    • rem : medida relativa de acordo com o tamanho da fonte que foi definida dentro do "body"
    • vw : medida relativa referente a largura da view-pot ou seja relativaa largura do seu monitor
    • vh : e similar ao "vw" o seu diferencial e que ao invez de definir a medida relativa de acordo com a largura ele defini de acordo com a altura

Obs: para definir tamanho das fontes no css e recomendado utilizar dois padrões ("px" e "em").
O tamanho da fonte pre definido pelo proprio navegar e de "16px" sendo assim o valor "1em corresponde a == 16px"
