Utilizando css externo

Basicamente a utilização do css externo a e mais utilizada pois esse metodo possibilita uma melhor compreenção do codigo sem falar na organição dos arquivos, ele e similar ao css interno ou seja a tag selecionada recebera a formatação independente se a varias tags do mesmo tipo todas irão receber a mesma formatação

Obs: No caso do css externo todas as formatação forma definidas dentro de um arquivo ".css" atravez desse arquivo realizamos uma chamada que possibilita que o html se comunique com as formatação desse arquivo, basicamente esse arquivo css e chamado dentro do cabeçalho do html atravez da tag "link:css" o ponto positivo de se utilizar a css exteno e que caso voce faço um site com 10 paginas pode se utilizar apenas um arquivo css para gerar assas formatações nas 10 paginas e se tiver a nescessidade de auterar alguma formatação basta ir no aruivo css e auterar
