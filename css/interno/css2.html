Utilizando css interno

Basicamente a css interno e similar ao css inline o seu diferencial e que a formatação não e definida no corpo da tag, nesse caso definimos as formatações a serem aplicadas dentro da cobeçalho do html ou seja dentro da tag "head" para relizar essas fomatações e nescessario chamar a tag "style" dentro dessa tag podemos gerar formataçõe no html ou seu difenrecia do css inline e que no css interno assim como no css externo pode-se definir formatações para tags nesse caso a tag definida para receber a formatação sera modificado ou seja todas as tags do mesmo tipo que estiverem contida dentro do html receberão a mesma formatação
