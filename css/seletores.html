Utilizando Seletores em Css

Basicamente os seletores em css tem um papel uito importante pois com eles podemos definir formatações de tags com mais facilidade
Um seletor css e utilizado quando se tem a nescessidade de realizar uma formatação em uma ou mais tags em espesifico ou seja em uma tag desejada
imagine o seguinte modelo, temos varias tags de paragrafo "p" e queremos realizar uma formatação em apenas uma ou mais tags desse tipo poremnão em todas no geral utilizariamos o css inline porem esse metodo não e recomendado, para isso utilizamos o css externo pois e o padrão mais utilizado e para selecionar a tag em expecifico utilizamos os seletores css.
Basicamente existe 4 maneiras de se definir um seletor css

1º id-(#) : Basicamente com esse seletor podemos definir formatações css em uma unica tag em expecifico

2º classe-(.) : Basicamente com esse seletor podemos definir formatações em uma ou mais tags definidas

3º Pseudo_Classe-(:) : As "Pseudo_Classe" basicamente são funcionalidades-(eventos) pre definidos que utilizamos para gerar certas formatações ou seja as pseudo classe ja estão pre-definidas basta definirmos uma formatção em espefico e assim que a funcionalidade/evendo dessa pseudo-classe for executado essa pseudo-classe ira realizar a formatação definida

4º Filho de um elemento-(>) : Basicamente dentro das css para definir uma formatação de um elemento que esteja contidentro dentro de outro elemento ou seja um elemento filho basta utilizar o parametro ">" esse parametro define que o elemento e filho de outro

Observa-se a diferença em realizar formatações css utilizando seletores ou não, observamos que a utilização dos seletores facilitam muito na hora de definirmos a formatação para um elemento em especifico

Obs: apesar de podemos definir um unico id e utilizar ele em varias tags o ideal e utilizar o seletor "id" em uma unica formatação ou seja em uma unica tag, e utilizar o seletor por "classe" quando tiver nescessidade de realizar formatações em varias tags sendo elas do mesmo tipo ou não.
Resumindo quando tiver a nescessidade de gerar formatação em um unico elemento utiliza-se o seletor por "id" caso tenha que gerar formatações em mais de um elemento utiliza-se o seletor po "classe"
