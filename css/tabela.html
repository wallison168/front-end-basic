Trabalhando com tabelas

Basicamente utilizamos tabelas dentro de um site para mostrarmos conteudos cadastrados como itens de uma loja ou coisas desse tipos, elementos que tem a nescessidade de ser inseridos dentro de uma tabela para facilitar pro usuario a compreender melhor o conteudo.

As tabelas possuem uma hierarquia a ser seguidas definidos por tags
1º table : basicamente esse e o escopo principal de uma tabela basicamente os componentes de uma tabela são definidos dentro dessa tag.

2º tr : basicamente essa tag definie a linha da tabela e deve ser definida dentro da tag "table" ou seja basicamente ela e uma tag filho da tag "table".

3º th : basicamente essa tag define o cabeçalho da tabela e deve ser definida dentro da tag "tr" ou seja ela e uma tag filho da tag "tr".

4º td : basicamente essa tag define o conteudo a ser inserido dentro da tabela e deve ser inserido dentro da tag "tr" ou seja ela e uma tag filho da tag "tr".

EX: Definido uma tabela basica

nome idade sexo
wallison 21 masculino

Estilização para tabelas

Existem algumas estilizações para tabelas para facilitar na formatação dos elementos.

1º border-collapse: = basicamente essa formatação e utilizada para evitar separações das bordas da tabela ou seja ao inserir uma borda na tabelo o html por padrão ja define o valor desse paramtro como "separate" e isso causa uma separação entre as bordas, para que as borda possão ficar juntas igual de uma tabela e nescessario utilizar o valor nesse parametro de "collapse" isso faz com que as bordas possão ficar juntas.
