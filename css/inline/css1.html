Utilizando Css Inline

Basicamente podemos definir as "css" de 3 maneiras distintas
1º inline: basicamente esse metodo trata-se de definirmos a css dentro da propria tag ou seja definir a formataçaão dentroa da linha de comando da tag
2º interno: basicamente esse metodo e definido dentro do proprio html assim com o metodo "inline" o seu diferencial e que definimos ele dentro do cabeçalho do htlm ou seja dentro da tag "head" nesse caso para definirmos essa formatação e nescessario chamar a tag "style" e dentro dela definir qual elemento sera formatado
3º externo: esse e o metodo padrão de se utilizar a css pelo fato de ser definido em um arquivo externo, isso facilita uma compreeção melhor do codigo e possiblita uma facil modificação do codigo ja que uma vez que ele foi definido fora do corpo do html pode-se identificar com facilidade o conteudo em especifico
