Trabalhando com Box Model

Basicamente Box Model e a representação dos container onde as informações contidas dentro da pagina estão armazenadas ou seja todas as tags são uma especie de "box (caixa)".

Basicamente existem dois tipos de "Box" ("Box-level" "Inline-level")

1º Box-Level : Basicamente o box-level e um tipo de container/cixa que possue um display em block ou seja não ha outras tags na mesma linha onde se encontra esse tipo de container, sendo assim todo o conteudo fora dessa caixa ficara uma linha a baixo dela relizando um efeito de quebra de linha.

2º Inline-Level : Ja o inline-level permite que conteudo seja colocados na mesma linha onde se encontra a caixa ou seja esse conteiner permite que conteudos inseridos fora dele possa fica na mesma linha onde se encontra.

Tags que representam o Box-Model

  1. Box-Level
    • div
    • h1...h6
    • p
    • main
    • header
    • nav
    • article
    • aside
    • footer
    • form
    • video
  2. Inline-Level
    • span
    • a
    • code
    • small
    • strong
    • em
    • sup..sub
    • label
    • button
    • input
    • select

Resumindo: Box-Level são todas as tags que possuem o display em block ou seja todo conteudo fora dela ficara uma linha a baixo recebendo um quebra de linha.
Ja o Inline-Level permite qeu outro conteudo seja colocados na mesma linha dele
EX:

Box_Level

Inline-Level : test

Obs: observa-se que o container do tipo "Box-Level" não permite que nenhum outro elemento que esteja fora possa ficar na mesma linha que ele.
Ja o container do tipo "Inline-Level" permite que isso aconteça

Definição de uma Caixa

O Box-model tem uma definição interna no css par definir o seu espaçamento basicamente são 3 propriedades para definir o espaçamento entre o elemento dentro da caixa e os demais elementos
1º padding : Basicamente essa propriedade define o espaçamente entre o conteudo inserido dentro da "Box" e a borda

2º border : E o intercessor entre a padding e a margin, basicamente ele defini o tamanho da box podendo tracejar uma linha definindo o seu tamanho exato para o usuario

3º margin : Basicamente o margin e um espaçamento que fica fora da "Box" ou seja ele define um espaçamento entre a borda da box e o proximo elementoa ser inserido dentro da pagina

Propriedades css para Box-Model

Propriedade Border
1º border-width : basicamente essa propriedade define a largura da borda

2º border-style : basicamente essa propriedade define o estilo da borda

3º border-color : basicamente essa propriedade define a cor da borda

Obs: Forma simplificada do operador Border

1º border : width style color = basicamente podemos definir as 3 formatações do operador "border" passando os seus valores na ordem correta.

Obs: deve-se seguir a ordem correta na hora de colocar os valores dentro da propriedade "border" ou seja o primeiro valor a ser inserido deve ser a largura da borda "width", o segundo valor a ser inserido deve ser o estilo da borda "style" e o ultimo valor a ser inserido deve ser a cor da borda "color"

Propriedade Padding
1º padding-top : basicamente essa propriedade define o espaçamento na parte superior do elemento

2º padding-right : basicamente essa propriedade define o espaçamento na parte direita do elemento

3º padding-bottom : basicamente essa propriedade define o espaçamento na parte inferior do elemento

4º padding-left : basicamente essa propriedade define o espaçamento na parte esquerda do elemento

Obs: Existem 3 formas simplificadas de se definir o Padding

1º padding : px/em = Basicamente utilizando apenas o operador "padding" e passando apenas um valor, esse valor sera definido para os quatro cantos.

2º padding : px px = Utilizando apenas o operador "padding" e passando dois valores, o primerio valor ira definir o espaçamento do canto inferior e superior ja o segundo valor ira definir o espaçamento das laterais direita e esquerda.

3º padding : px px px px = Pode-se se usar o operador "padding" passando quatro valores definindo os quatro cantos, o primeiro valor irada definir o espaçamento do canto superior, o segundo valor ira definir o do canto direito, o terceiro ira definir a parte inferior e o ultimo valor vai definir a parte esquerda.

Propriedade Margin
1º margin-top : basicamente essa propriedade define o espaçamento na parte superior do elemento

2º margin-right : basicamente essa propriedade define o espaçamento na parte direita do elemento

3º margin-bottom : basicamente essa propriedade define o espaçamento na parte inferior do elemento

4º margin-left : basicamente essa propriedade define o espaçamento na parte esquerda do elemento

Obs: Existem funcionalidade especiais para o operador "margin" formas simplificadas

1º margin: auto = Utilizando apenas a propriedade "margin" com o valor "auto" isso faz com que centralize a box horizontalmente ou seja suas laterais

2º margin: px px px px = Pode-se se usar o operador "margin" passando quatro valores definindo os quatro cantos, o primeiro valor irada definir o espaçamento do canto superior, o segundo valor ira definir o do canto direito, o terceiro ira definir a parte inferior e o ultimo valor vai definir a parte esquerda.

3º margin: px px = Utilizando apenas o operador "margin" e passando dois valores, o primerio valor ira definir o espaçamento do canto inferior e superior ja o segundo valor ira definir o espaçamento das laterais direita e esquerda.

4º margin: px = Basicamente utilizando apenas o operador "margin" e passando apenas um valor, esse valor sera definido para os quatro cantos.

Aplicando sombra na Box
Basicamente para aplicar sombras em uma "Box" utilizamos o seguinte parametro "box-shadow" atravez desse parametro css que definimos a sombra na box.

1º box-shadow : px px px color = Basicamente esse parametro recebe 4 valores dentre eles 3 desses valores são numericos que irão definir ("Deslocamento-horizontal" "Deslocamento-vertical" "Espalhamento") e por ultimo ira definir a cor da sombra da box.

Obs: Existe um valor padrão para definir a sombra de uma box "box-shadow: 1px 1px 1px black;"

Arredondando a borda da box
Basicamente para aplicar o Arredondamento na borda da box e nescessario usar o seguinte paramtro "border-radius" atarvez desse parametro e que definimos qual vai ser o Arredondamento da borda da box.

1º border-radius : px = Basicamente esse parametro rece um valor em px/em que ira definir o arredondamento da box, quanto maior for o valor maior sera o arredondamento da borda da box.

Obs: Pode se definir cada canto arredondado separadamente

1º border-top-left-radius: px = basicamente esse parametro recebe um unico valor definindo o arredondamento da borda no canto "superior esquerdo"

2º border-top-right-radius: px = basicamente esse parametro recebe um unico valor definindo o arredondamento da borda no canto "superior direito"

3º border-bottom-right-radius: px = basicamente esse parametro recebe um unico valor definindo o arredondamento da borda no canto "inferior direito"

4º border-bottom-left-radius: px = basicamente esse parametro recebe um unico valor definindo o arredondamento da borda no canto "inferior esquerdo"

5º border-radius : px px px px = pode se definir o parametro "border-radius" passando os 4 valores de cada canto da borda da box, sendo o primeiro valor representando o canto superior esquerdo, o segundo referente ao conto superiror direito, o terceiro valor referente ao canto inferior direito e o ultimo valor representa o canto inferior esquerdo. Seguindo a mesma ordem separadamente definida logo acima.
