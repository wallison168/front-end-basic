Trabalhando com Box Model

Basicamente Box Model e a representação dos container onde as informações contidas dentro da pagina estão armazenadas ou seja todas as tags são uma especie de "box (caixa)".
Basicamente existem dois tipos de "Box" ("Box-level" "Inline-level")
1º Box-Level : Basicamente o box-level e um tipo de container/cixa que possue um display em block ou seja não ha outras tags na mesma linha onde se encontra esse tipo de container, sendo assim todo o conteudo fora dessa caixa ficara uma linha a baixo dela relizando um efeito de quebra de linha.

2º Inline-Level : Ja o inline-level permite que conteudo seja colocados na mesma linha onde se encontra a caixa ou seja esse conteiner permite que conteudos inseridos fora dele possa fica na mesma linha onde se encontra.

Tags que representam o Box-Model

  1. Box-Level
    • div
    • h1...h6
    • p
    • main
    • header
    • nav
    • article
    • aside
    • footer
    • form
    • video
  2. Inline-Level
    • span
    • a
    • code
    • small
    • strong
    • em
    • sup..sub
    • label
    • button
    • input
    • select

Resumindo: Box-Level são todas as tags que possuem o display em block ou seja todo conteudo fora dela ficara uma linha a baixo recebendo um quebra de linha.
Ja o Inline-Level permite qeu outro conteudo seja colocados na mesma linha dele
EX:

Box_Level

Inline-Level : test

Obs: observa-se que o container do tipo "Box-Level" não permite que nenhum outro elemento que esteja fora possa ficar na mesma linha que ele.
Ja o container do tipo "Inline-Level" permite que isso aconteça

Definição de uma Caixa

O Box-model tem uma definição interna no css par definir o seu espaçamento basicamente são 3 propriedades para definir o espaçamento entre o elemento dentro da caixa e os demais elementos
1º padding : Basicamente essa propriedade define o espaçamente entre o conteudo inserido dentro da "Box" e a borda

2º border : E o intercessor entre a padding e a margin, basicamente ele defini o tamanho da box podendo tracejar uma linha definindo o seu tamanho exato para o usuario

3º margin : Basicamente o margin e um espaçamento que fica fora da "Box" ou seja ele define um espaçamento entre a borda da box e o proximo elementoa ser inserido dentro da pagina

Valores da propriedade "border" em css

1º border-width : basicamente essa propriedade define a largura da borda

2º border-style : basicamente essa propriedade define o estilo da borda

3º border-color : basicamente essa propriedade define a cor da borda
